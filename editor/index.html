Instructions
Each Enter creates a new paragraph.
Select some text using mouse, keyboard or double click on a word and a tooltip will show up.
To rearrange the paragraphs, click on Done button and just drag the paragraphs.
Links can be created using anchor tags.These links will be displayed on clicking Done Button in alternating red and blue colors.
To replace all the four letter words present, click the replace button in the navigation bar.To distinguish the replaced words you can hover over them and their background color changes.
The content is saved as the user types and will be available even after reload.To clear the editor Clear Button in the navigation bar can be used.
